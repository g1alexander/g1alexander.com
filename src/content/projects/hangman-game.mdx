Construí este proyecto utilizando Next.js, OpenAI y reCAPTCHA. Aprendí a manejar la carpeta 'api' en Next.js para crear una API REST, y también experimenté con OpenAI para crear prompts personalizados y ajustar los parámetros para obtener respuestas más precisas. Además, implementé IndexedDB para guardar el progreso del juego, así que si alguien no ha completado una palabra, puede retomar donde lo dejó al volver o recargar la página. Ahora, mi objetivo es hacer que el juego sea más interactivo y que se pueda jugar completamente con el teclado.
