Solución de un reto de frontendmentor.io, en donde se plantea la construcción de un sitio web en el cual se puedan mostrar los datos y la dirección en donde se encuentra alojado un dominio.

La información de la IP o dominio se obtuvo a través de la API de IP Geolocation API y para mostrar la dirección se implementó el mapa de leafletjs. Las tecnologías implementadas fueron Nuxt JS junto con Vuex y el proyecto fue desarrollado por <a href="https://www.linkedin.com/in/g1alexander/" target="_blank" rel="noreferrer">Alexander Granados</a> y <a href="https://www.linkedin.com/in/felipevargass/" target="_blank" rel="noreferrer">Felipe Vargas</a>.
